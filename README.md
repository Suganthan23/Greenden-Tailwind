PROJECT TITLE: "Greenden Plant Store Front-End Clone"

DESCRIPTION:
A clean, responsive front-end clone of a fictional plant store landing page called Greenden, developed entirely using Tailwind CSS. The project focuses on layout precision, modern UI design, and mobile-first responsiveness.

The clone includes:

✅ Sticky navigation bar with logo and page links (Home, Products, Contact)
✅ Hero section with promotional text, call-to-action buttons, and image
✅ Informational "About" section with service highlights and icons
✅ Interactive best-seller product grid with hover effects
✅ Customer review section styled with testimonials and branding colors
✅ Newsletter subscription input and a branded footer

This project enhanced my understanding of utility-first CSS with Tailwind, responsive design principles, and interactive layout techniques without writing custom CSS.

TECHNOLOGIES USED:

HTML5

Tailwind CSS (CDN)

SVG Icons

Responsive Grid & Flex Layouts

RESPONSIVENESS:
Fully responsive across mobile, tablet, and desktop screen sizes using Tailwind’s built-in utility classes, with adaptive flex/grid layouts and mobile-first design structure.
